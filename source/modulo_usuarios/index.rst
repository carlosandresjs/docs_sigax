Módulo de Usuarios 
================================

Incluye todos los submódulos para la administración de usuarios, grupos de permisos y permisos por usuario.

======================
Submódulo de Grupos
======================

El sistema permite la creación de grupos de usuarios, esto hará que los usuarios con dichos grupos tengan permisos a distintas funciones o modulos dentro de la aplicación.

A continuación se listan cada una de las acciones habilitadas en este submódulo.
Para ver el detalle, hacer click sobre la acción de interés.

.. toctree::
    :maxdepth: 2

    grupos/crear
    grupos/editar
    grupos/eliminar
    grupos/basedatos

========================
Submódulo de Permisos
========================

El sistema permite la creación de permisos únicamente a través de comandos y por fuera de la aplicación, estos permisos posteriormente podrán ser asignados para habilitar o restringir el acceso de los usuarios.
Ingresar para ver el detalle de este submódulo.
